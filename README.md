# Simple Interest Calculator

A simple, user-friendly script designed to calculate the simple interest based on the principal amount, annual interest rate, and time period in years.

## Formula Used

The application uses the standard simple interest formula:

A = P * r * t

Where:
- **P** = Principal amount
- **r** = Annual interest rate (in decimal form)
- **t** = Time period (in years)

---

## Input Fields

To use the calculator, you will need to provide the following inputs:
1. **Principal Amount:** The initial sum of money invested or borrowed.
2. **Rate of Interest:** The annual interest percentage (e.g., enter 5 for 5%).
3. **Time Period:** The duration for which the money is invested or borrowed, specified in years.

---

## Output

- **Simple Interest:** The total interest earned or accrued over the specified time period.
